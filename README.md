# kyotocabient-node
kyotocabinet bindings for Node.js


# Requirements

## OS

- Linux
- Mac OS X

## modules

- node-gyp


# Building

    $ node-gyp configure build


# Installation

    $ npm install kyotocabient-node


# Running Tests

kyotocabient-node depends on [Vows](http://vowsjs.org/) for testing.

    $ vows test/*.js
